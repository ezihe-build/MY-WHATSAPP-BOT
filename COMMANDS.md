## Quick Tips

- Use `.menu` to see all available menus
- Reply to messages with commands like `.sticker`, `.toimg`, `.vv`
- Most commands have a 3-second cooldown
- Owner commands require your number to be set as owner
- Add API keys to `.env` for enhanced AI features
